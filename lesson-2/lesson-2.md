# Lesson 02 - Primitive Data Types Overview

## Topics Covered

- What are primitive data types?
- Go is a statically typed language.
- Go is a strongly typed language.
- Primitive vs Composite data types.
- Overview of basic data types:
  - bool
  - string
  - int
  - float
  - complex
- Zero values (overview only)

---

## Sample Variable Declarations

```go
var isName bool
var name string
var age int
var balance float64
var complexNum complex64
```

---

## Key Notes

- Every variable has a data type.
- Go determines data types at compile time.
- Primitive data types are the basic building blocks of Go.
- Composite data types are created using primitive data types.
- Every data type has a default (zero) value.
- Variables and each primitive data type will be covered in detail in upcoming lessons.

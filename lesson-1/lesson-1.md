# Lesson 01 - Getting Started

## Topics Covered

- Installing Go
- Creating a Go project
- `package main`
- `main()` function
- Importing packages
- `fmt.Println()`
- Running a Go program
- Building an executable

---

## Commands Learned

```bash
go run main.go
```

```bash
go build -o hello_world main.go
```

---

## Key Notes

- `main()` is the entry point of an executable Go program.
- `package main` makes the program executable.
- `fmt` is the standard formatting package.
- `Println()` prints output followed by a newline.
- `go run` compiles and executes the program.
- `go build` creates a standalone executable.
